This is the landing page for composer and theorist Jeffrey Martin's composition and analysis software. All software is available under the GNU GPLv3.0, and links to source code can be found on the pages for the software.

pctheory is a Python library for working with atonal theory. It has many features and allows composers and theorists to develop custom composition, analysis, and research programs without investing the time and effort to develop a standard theory library.

SetManipulator is a text-based program that makes atonal theory calculations quick and easy.
